Проведение мероприятий

Для других мероприятий, не требующих танцевальной площадки возможно увеличение посадочных мест до 50. С удовольствием примем у себя в гостях различные тренинги, тимбилдинг и другие развивающие и объединяющие мероприятия. Тем более, что для проведения тренингов есть проектор и телевизоры находящиеся над банкетным залом, для того чтобы гости лучше видели программу проведения.

  • Рядом с кафе «Буфет Обед» есть круглосуточная парковка по правому или левому ряду, машиномест хватит даже на очень большое количество гостей.

  • «Алкоголь с собой». Для того, чтобы избежать дополнительных трат, весь алкоголь можно закупить заранее. Заранее закупленный алкоголь можно использовать не только для проведения свадеб, но и для проведения банкетов, корпоративов, похорон, поминок и других мероприятий.
